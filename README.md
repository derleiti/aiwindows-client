# AIWindows Client

> PyQt6 Desktop-App für Windows 10/11 — AILinux AI-Backend Zugang

![AILinux](https://img.shields.io/badge/AILinux-Backend-blue)
![Python](https://img.shields.io/badge/Python-3.12-blue)
![PyQt6](https://img.shields.io/badge/PyQt6-GUI-green)
![Platform](https://img.shields.io/badge/Platform-Windows%2010%2F11-lightblue)

## Features

- **AI Chat** — 659+ Modelle via TriForce Backend
- **PowerShell Terminal** — integriertes Terminal-Panel
- **File Browser** — Datei-Manager mit AI-Analyse
- **MCP Tools** — direkter Zugang zu Backend-Tools
- **Multi-Model** — Modell-Wechsel on-the-fly
- **JWT Auth** — sicherer Login via login.ailinux.me

## Install

```powershell
# Installer herunterladen von:
# https://ailinux.me/downloads/

# Oder aus Source:
pip install -r requirements.txt
python main.py
```
